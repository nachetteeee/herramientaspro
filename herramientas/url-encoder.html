Preguntas frecuentes

¿Qué es la codificación de URL?

Es el proceso de convertir caracteres que no están permitidos en URLs (como espacios, ñ, tildes o &) al formato %XX, donde XX es el código hexadecimal del carácter según el estándar RFC 3986.

¿Cuándo necesito codificar una URL?

Cuando construyes URLs con parámetros que contienen texto libre, caracteres especiales o idiomas con caracteres no ASCII. Es obligatorio para que las URLs funcionen correctamente.

¿Qué diferencia hay entre encodeURI y encodeURIComponent?

encodeURI codifica una URL completa preservando los caracteres estructurales (:, /, ?, #). encodeURIComponent codifica todo excepto letras, números y - _ . ~ y se usa para valores de parámetros.

¿Se puede decodificar cualquier URL?

Sí, cualquier URL con codificación %XX estándar puede decodificarse. Esta herramienta admite tanto el formato encodeURI como encodeURIComponent.
