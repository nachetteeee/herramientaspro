¿Cómo usar esta herramienta?

  1. Pega tu JSON en el área de texto izquierda.
  2. El validador detecta errores automáticamente y los muestra resaltados.
  3. Si el JSON es válido, pulsa Formatear para obtener el resultado indentado.
  4. Copia el JSON formateado con el botón Copiar.

Preguntas frecuentes

¿Qué es JSON y por qué validarlo?

JSON (JavaScript Object Notation) es un formato ligero de intercambio de datos. Validarlo asegura que la sintaxis sea correcta antes de usarlo en una API o aplicación, evitando errores en producción.

¿Qué errores detecta el validador JSON?

Detecta comas faltantes o sobrantes, llaves y corchetes sin cerrar, comillas incorrectas, valores no válidos y otros errores de sintaxis comunes según el estándar RFC 8259.

¿Se envía mi JSON a algún servidor?

No. Toda la validación y formateo ocurre en tu navegador. Tu JSON nunca sale de tu dispositivo.

¿Cuál es la diferencia entre JSON válido y JSON bien formateado?

Un JSON válido cumple la sintaxis estrictamente. Un JSON bien formateado además está indentado y estructurado para ser legible por humanos. Esta herramienta hace ambas cosas.
