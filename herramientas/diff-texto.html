¿Cómo usar esta herramienta?

  1. Pega el texto original en el panel izquierdo.
  2. Pega la versión modificada en el panel derecho.
  3. Las diferencias se resaltan automáticamente: verde para adiciones, rojo para eliminaciones.
  4. Revisa cada cambio línea a línea para comparar las versiones.

Preguntas frecuentes

¿Para qué sirve el diferenciador de texto?

Para comparar dos versiones de un texto y ver exactamente qué cambió. Es muy útil para revisar ediciones de documentos, comparar versiones de código, detectar cambios en contratos o cualquier par de textos.

¿Cómo funciona el algoritmo de diferencias?

Usa el algoritmo LCS (Longest Common Subsequence) que identifica la secuencia común más larga entre dos textos y marca lo que fue añadido o eliminado para pasar de uno al otro.

¿Puedo comparar código de programación con esta herramienta?

Sí. Funciona con cualquier texto plano, incluyendo código. Para proyectos de software, herramientas como git diff ofrecen comparación más avanzada con contexto de repositorio.

¿Los textos comparados se envían a algún servidor?

No. La comparación ocurre completamente en tu navegador. Tus textos no salen de tu dispositivo.
